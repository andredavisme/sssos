# SSSOS

**Systems for Systematic Systems — An Observation of Systems**  
Volume: *The Business of Life*

A free, zero-backend, interactive education platform hosted on GitHub Pages.

## Live Site
[https://andredavisme.github.io/sssos](https://andredavisme.github.io/sssos)

## Features
- Single-page app — no framework, no build step, no backend
- Progress tracked entirely via `localStorage` (survives page reloads, no account needed)
- Responsive — works on mobile and desktop
- Personal reflection fields auto-save as you type
- Section 7 consolidates all reflection entries for final review

No build step needed. Open `index.html` directly in a browser, or serve with any static file server:
```bash
npx serve .
```
